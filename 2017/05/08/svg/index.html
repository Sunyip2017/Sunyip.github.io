svg

发表于 2017-05-08 | 分类于 svg | |

为什么要学svg？为了让页面更炫！

Step1、声明SVG视口

属性version和属性baseProfile属性是必不可少的，供其它类型的验证方式确定SVG版本。
指定了宽高为100像素的区域，不带单位，默认为px，也可以使用其他合法单位；
默认设置了一个坐标系，左上角为原点，其中水平(x)坐标向右递增,垂直(y)坐标乡下递增。

这里定义的画布尺寸是200200px。但是，viewBox属性定义了画布上可以显示的区域：从(0,0)点开始，100宽100高的区域。这个100100的区域，会放到200200的画布上显示。于是就形成了放大两倍的效果。

Step2、了解API

rect 矩形

svg默认宽高为300px、150px;

stroke相当于border-color；
fill相当于background-color;
还可以定义stroke-width边框的宽度；
rx、ry 圆角效果；

圆形

椭圆只要将rx与ry设置成不一样即可：

圆和椭圆有一个有用的特性就是可以用来做热点链接：

groups 分组

简写为<g></g>,类似于div，g中定义的样式，在其子元素中可以继承。

也可以为g设置id或者class，在style定义样式；

use标签

假如页面中有好几个圆形，这时候用use就非常方便了；
use可以引用页面中已经定义好的图形，然后改变图形的相关属性达到服用的效果；

defs 预先隐藏

隐藏不显示的图形，要使用的话用use引用；

line 线条

x1、y1为起点的x、y位置，x2、y2为结束点的x、y位置

polyline 折线

points
点集数列。每个数字用空白、逗号、终止命令符或者换行符分隔开。每个点必须包含2个数字，一个是x坐标，一个是y坐标。所以点列表 (0,0), (1,1) 和(2,2)可以写成这样：“0 0, 1 1, 2 2”。

playgon 多边形

points
点集数列。每个数字用空白符、逗号、终止命令或者换行符分隔开。每个点必须包含2个数字，一个是x坐标，一个是y坐标。所以点列表 (0,0), (1,1) 和(2,2)可以写成这样：“0 0, 1 1, 2 2”。路径绘制完后闭合图形，所以最终的直线将从位置(2,2)连接到位置(0,0)。

path 最强大的

路径(字母大写是绝对位置，小写是相对位置，相对于前一个点移动多少距离)：
M x y 起点 或者 m dx dy
L x y 画线 或者 l dx dy 定义在最后 x、y指向起点
H x 平行线 或者 h dx
V y 垂直线 或者 v dy
Z 返回起点 或者 z
